# Aldric Frontend

React/Vite dashboard for Aldric AI Trading Assistant.

## Setup

```bash
cd frontend
npm install
cp .env.example .env
npm run dev
```

Set `VITE_API_BASE_URL=http://localhost:5000/api` in `.env`.

## Features

- BTC/USDT market overview
- Candlestick-derived price chart
- Risk settings panel
- Signal generation button
- Latest signal card
- Signal history table
- Visible trading risk disclaimer
